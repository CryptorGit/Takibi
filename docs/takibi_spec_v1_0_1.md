# 焚き火パズル v1.0.1 仕様書（完全版）

## 0. コンセプト
“焚き火の前で静かに串を作り、ゆっくり焼けるのを眺めながら、素材管理のパズルを解くチルゲーム。”

- 操作ストレスゼロ
- 時間制限ゼロ
- アクション操作ゼロ
- 癒しの焚火×キャラ
- パズル性は「手札管理」「スロット管理」「素材組み合わせ」
- 提供は自動
- ゲームオーバーは手札があふれた時のみ

## 1. ゲームループ（ターン制）
### ターンの流れ
1. 素材が3個ドローされる（手札に追加）
2. 素材を串に刺して組み立てる（任意）
3. 串を焚火スロットへ置く（任意）
4. ターン終了ボタン→焼きターン減算
5. 焼きターン0の串は自動提供→スコア加算

---

## 2. 素材データ仕様（v1.0.1）

| 素材       | 焼きターン/個 | スコア/個 | 出現確率 |
|-----------|---------------|-----------|----------|
| マシュマロ | 0.4           | 1         | 30%      |
| ウインナー | 1.0           | 2         | 30%      |
| 団子       | 1.0           | 3         | 25%      |
| 豚バラ     | 1.2           | 4         | 15%      |

### 焼きターン計算
```
合計 = Σ(素材ごとの焼きターン)
実焼きターン = floor(合計)
```

---

## 4. スコア計算仕様
### 基本スコア
Σ(素材スコア × 個数)

### コンボボーナス
同一素材が連続2個以上：
```
(個数 - 1) × 素材スコア
```

---

## 5. ゲームオーバー条件
手札が上限20個を超えた瞬間、ゲームオーバー。
